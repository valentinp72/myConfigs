# myVIM
My vimrc config file + plugins

## How to install:

Copy and paste this into a terminal:

```shell
mkdir -p ~/git/myVIM &&\
git clone https://github.com/valentinp72/myVIM.git ~/git/myVIM &&\
~/git/myVIM/install.sh
```
## How to update:

Copy and paste this into a terminal:

```shell
git pull ~/git/myVIM/ &&\
~/git/myVIM/install.sh TRUE
```

## How to remove:

Copy and paste this into a terminal:

```shell
rm -rf ~/git/myVIM
```
